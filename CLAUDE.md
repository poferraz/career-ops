# career-ops

## Session Management: Tracking System

At the START of every session:
1. Read tracking/state.md — summarize where we left off
2. Read tracking/tasks.md — show the current priority tasks
3. Check tracking/decisions.csv — flag any decisions past their review date
4. Recommend what to work on next based on task priority

During the session:
- When I make a decision, log it to tracking/decisions.csv with: today's date, decision, reasoning, expected outcome, 30-day review date, status "active"
- When I complete a task, move it to Done in tracking/tasks.md with today's date
- When new tasks come up, add them to tracking/tasks.md at the appropriate priority

At the END of every session:
1. Update tracking/state.md with what we did and what to pick up next
2. Update tracking/tasks.md to reflect current state
3. Verify any new decisions are in decisions.csv
