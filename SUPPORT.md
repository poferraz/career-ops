# Support

## Getting Help

| Channel | Use for |
|---------|---------|
| [GitHub Issues](https://github.com/poferraz/career-ops/issues) | Bug reports, unexpected behavior, broken output |
| [GitHub Discussions](https://github.com/poferraz/career-ops/discussions) | Questions, ideas, platform integration help |
| [SECURITY.md](SECURITY.md) | Private vulnerability reports — do NOT use Issues for security |

## Before Opening an Issue

1. Check the [README](README.md) — install and platform compatibility are covered there.
2. Check [CHANGELOG.md](CHANGELOG.md) — your issue may already be fixed in a newer version.
3. Check [existing issues](https://github.com/poferraz/career-ops/issues) — avoid duplicates.

## Response Timeline

This is a single-maintainer project. Expect responses within **3–5 business days**.
Security reports are prioritized and typically acknowledged within **48 hours**.

## Contributing a Fix

If you've identified the root cause, a PR is faster than an issue.
See [CONTRIBUTING.md](CONTRIBUTING.md) for the process.
